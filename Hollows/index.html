.- Hollows Of Rogue -.

"Rogue, but better!"

Descent into the deadly dungeon, fight monsters, find loot and bring back the legendary Amulet of Rogue! An adaptation of an ASCII text terminals classic from '80s which gave the genre its name, greatly enhanced and streamlined for easier and more fun gameplay on modern devices.

Touch screen friendly

Swipe to move around, pinch to zoom, drag to pan around the dungeon. Use all hero commands via touch optimized modal interactions. Easy inventory management and new command to navigate automatically to the exit included.

Adjustable difficulty

The original game was not easy and was quite punishing - some original game features can be selectively turned off to allow for more forgiving, but hopefully more fun runs! There are more changes too numerous to list, but there's also no hunger mechanic.
