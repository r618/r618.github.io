"Rogue, but better!"

An adaptation of old ASCII text terminals game from '80s which gave the genre its name, remade for new era.

Optimized for touch screens

Swipe to move, all commands via touch optimized modal interactions. Plus some new ones.

Adjustable difficulty

The original game was hard and extremely punishing - some original game features can be turned off, and there's no hunger mechanic.
